# Запуск Лабораторної роботи №7: Reactive Todo List

Цей проєкт використовує TypeScript та RxJS. Для перевірки роботи необхідно скомпілювати код.

### Інструкція із запуску:

# 1. Перейдіть у головну папку репозиторію (asia-web)
cd asia-web

# 2. Встановіть необхідні залежності (TypeScript та RxJS)
npm install

# 3. Скомпілюйте TypeScript-код у JavaScript
npm run build:lab7-todolist

# 4. Запуск у браузері
Відкрийте файл `lab7-todolist/index.html` за допомогою розширення Live Server.
