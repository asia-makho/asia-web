# 🌍 Weather & Time Dashboard | Personal Portfolio

Hi there! 👋 This is my interactive portfolio designed to showcase modern web development skills. What started as a simple static page has evolved into a fully dynamic, responsive web application featuring real-time data fetching and state management.

## 🔗 Live Demo
**https://asia-makho.github.io/asia-portfolio-web/** 
## 🚀 Key Features

* **⛅ Dynamic Weather Dashboard:** Integrated with the OpenWeatherMap API using asynchronous JavaScript (`Fetch API`, `async/await`) to display real-time temperature, humidity, and wind speed for any city.
* **🕒 Live Analog Clock:** Built entirely with Vanilla JavaScript and CSS transformations for smooth, real-time hand movement.
* **🌓 Dark/Light Mode:** A user-friendly theme toggle with preferences saved locally using `localStorage` for a seamless, persistent experience across sessions.
* **🌐 Bilingual Ready:** Navigation and UI structure prepared for seamless switching between English and Ukrainian.
* **✨ Modern UI/UX:** Designed in a "Soft Minimalism" style, featuring fluid keyframe animations, responsive CSS grids, and a clean layout.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox, CSS Grid, Keyframe Animations, Custom Properties).
* **JavaScript (ES6+):** * Asynchronous programming (Fetch API)
  * DOM Manipulation & Event Handling
  * Web Storage API (`localStorage`)
  * Timers (`setInterval`) & Date Object math
* **API:** OpenWeatherMap API

## 📸 Project Structure

The project consists of three main sections:
1. **Home:** Information regarding my education, background, and technical skills.
2. **Weather & Time:** A functional dashboard featuring the live widgets.
3. **Contacts:** Quick and easy links to get in touch with me, featuring animated social media icons.
